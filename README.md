# Постановка задачи и аналитика

Для падения поле рассеяния метаповерхности может быть представлено в общем в виде следующим образом 

$$
E(\theta, \varphi) = \sum_{m=0}^{M-1} \sum_{n=1}^{N-1} A_{mn} e^{j\alpha_{mn}} f_{mn}(\theta, \varphi) B_{mn} e^{j\varphi_{mn}}
e^{jk_0(md_x\sin\theta \cos \varphi + n d_y \sin \theta \sin \varphi)}
$$

$A_{mn}, \alpha_{mn}$ - амплитуда и фаза падающего поля 

$B_{mn}, \varphi_{mn}$ - амплитуда и фаза отраженного поля 

$f_{mn}(\theta, \varphi)$ - ДН ячейки 

$d_x, d_y$ - периодичность в направлениях $x$ и $y$

Для микрополосковой патч-ячейки предполагаем $f_{mn}(\theta, \varphi) = \cos \theta$

$$
E(\theta, \varphi) = \cos \theta \sum_{m=0}^{M-1} \sum_{n=1}^{N-1} A_{mn} e^{j\alpha_{mn}} B_{mn} e^{j\varphi_{mn}}
e^{jk_0(md_x\sin\theta \cos \varphi + n d_y \sin \theta \sin \varphi)}
$$

Для простоты считаем, $A_{mn}=B_{mn}=1$

$$
E(\theta, \varphi) = \cos \theta \sum_{m=0}^{M-1} \sum_{n=1}^{N-1} e^{j\alpha_{mn}} e^{j\varphi_{mn}}
e^{jk_0(md_x\sin\theta \cos \varphi + n d_y \sin \theta \sin \varphi)}
$$

В нашем случае, фаза принимает два значения, так что мы можем закодировать $\varphi_{mn}$ как '1' и '0'. 
Тогда бинарная кодирующая матрица будет иметь вид 

$$
\Phi = 
\begin{pmatrix}
1/0 & 1/0 & \cdots & 1/0 \\ 
1/0 & 1/0 & \cdots & 1/0 \\ 
\vdots & \vdots & \ddots & \vdots \\ 
1/0 & 1/0 & \cdots & 1/0 
\end{pmatrix}
$$

Для оптимизации такого вида матрица, очень удобно ложится Генетический алгоритм 

## Ускорение вычислений с помощью БПФ

Прямой расчет поля по формуле двойной суммы требует $O(MN)$ операций для одного направления и $O((MN)^2)$ для полной картины рассеяния, что крайне неэффективно для больших апертур. Для ускорения используем двумерное дискретное преобразование Фурье (2D-БПФ).


$$
E(\theta, \varphi) = \cos \theta \sum_{m=0}^{M-1} \sum_{n=1}^{N-1} e^{j\alpha_{mn}+j\varphi_{mn}}
e^{jk_0(md_x\sin\theta \cos \varphi + n d_y \sin \theta \sin \varphi)}
= \cos \theta \sum_{m=0}^{M-1} \sum_{n=1}^{N-1} W_{mn}
e^{jk_0(md_x\sin\theta \cos \varphi + n d_y \sin \theta \sin \varphi)}
$$

Введем пространственные частоты волны: 


$$
\begin{aligned}
u &= \frac{d_x}{\lambda} \sin \theta \cos \varphi  \\
v &= \frac{d_y}{\lambda} \sin \theta \sin \varphi 
\end{aligned}
$$

В итоге 

$$
\begin{aligned}
W_{mn} &= e^{j\varphi_{mn}} e^{j \alpha_{mn}} \\
E(u, v) &= \cos \theta \sum_{m=0}^{M-1} \sum_{n=0}^{N-1} W_{mn} e^{j 2\pi (mu+nv)}
\end{aligned}
$$

Дискретизируем углы 

$$
\begin{gather*}
u_k = \frac{k}{M} \\
v_l = \frac{l}{N} \\
E(k, l) = \cos \theta \sum_{m=0}^{M-1} \sum_{n=0}^{N-1} e^{j(\varphi_{mn} + \alpha_{mn})}e^{j2\pi[\frac{mk}{M} + \frac{nl}{N}]}= \\
= \cos \theta \sum_{m=0}^{M-1} \sum_{n=0}^{N-1} e^{j(\varphi_{mn} + \alpha_{mn})}e^{j2\pi[\frac{mk}{M} + \frac{nl}{N}]}\\
= \cos \theta \cdot IFFT2\{ e^{j(\varphi_{mn} + \alpha_{mn})}\} \quad \quad (1) 
\end{gather*}
$$

Посмотрим внимательнее чему равна фаза падающей волны $\alpha_{mn}$

Пусть падающая плоская волна приходит под углом $(\theta_0, \varphi_0)$. Тогда 

$$
\alpha_{mn} = k_0 (md_x \sin \theta_0) \cos \varphi_0 + n d_y \sin \theta_0 \sin \varphi_0)
$$ 

или через пространственные частоты падающей волны 

$$
\begin{gather*}
u_0 = \frac{d_x}{\lambda}\sin\theta_0 \cos \varphi_0 \\
v_0 = \frac{d_y}{\lambda} \sin \theta_0 \sin \varphi_0 
\end{gather*}
$$

Тогда 

$$
\alpha_{mn} = 2\pi (mu_0 + nv_0)
$$

# `class ScatteringCalculator`

### `def calculate_far_field_with_source`


Определяем задержку фаз для каждой ячейки $\varphi_{mn}$

```python
phase_reflection = np.where(coding_matrix == 0, np.pi, 0.0)
```

Считаем пространственные частоты падающей волны и фазу падающей волны $\alpha_{mn}$

```python
u_source = np.sin(theta_source) * np.cos(phi_source)
v_source = np.sin(theta_source) * np.sin(phi_source)

phase_source = - self.k0 * (self.x_grid * u_source + self.y_grid * v_source)
```

Вычисляем $W_{mn}$ 

```python
total_phase = phase_reflection + phase_source 
aperture_field = self.amplitude * np.exp(1j * total_phase)
```

Через БПФ считаем сумму из формулы (1)

```python
scattering = fftshift(ifft2(aperture_field)) 
```

Эти строчки соответствуют вычислению $f_{mn}(\theta, \varphi)$ - ДН ячейки 

```python
element_pattern = np.zeros_like(self.theta_grid, dtype=float)
element_pattern[self.visible_mask] = np.cos(self.theta_grid[self.visible_mask])
```

Итоговая формула для поля как произведение множителя решетки и ДН ячейки 

```python
far_field = scattering * element_pattern 
far_field[~self.visible_mask] = 0.0 
```

### `def calculate_directivity`

Функция для вычисления КНД по заданному полю в дальней зоне 

Замечание: рассматриваем только одну полуплоскость 

$$
\begin{gather*}
D = \frac{4 \pi}{P_{rad\_norm}} \\ 
P_{rad} = \int_{0}^{2\pi} \int_0^{\pi/2} F^2(\theta, \varphi) \sin \theta d \theta d \varphi
\end{gather*}
$$

Делаем замену переменных и считаем якобиан 

$$
\begin{gather*}
u = \sin \theta \cos \varphi \\
v = \sin \theta \sin \varphi \\ 
J = \frac{\partial (u, v)}{\partial (\theta, \varphi)}
= 
\begin{vmatrix}
\cos \theta \cos \varphi & - \sin \theta \sin \varphi \\
\cos \theta \sin \varphi & \sin \theta \cos \varphi 
\end{vmatrix}
= \cos \theta \sin \theta \cos^2 \varphi + \cos \theta \sin \theta \sin^2 \varphi 
= \cos \theta \sin \theta 
\end{gather*}
$$

Таким образом для дискретного случая можем посчитать КНД по следующей формуле 

$$
\begin{gather*}
P_{rad} = \sum_{l=0}^{N-1} \sum_{k=0}^{M-1} F^2(\theta_{kl}, \varphi_{kl}) \frac{du dv}{\cos \theta_{kl}} \\
D = \frac{4 \pi}{P_{rad\_norm}} 
\end{gather*}
$$

Эта формула и реализована в методе `def calculate_directivity`

# Генетический алгоритм

Генетический алгоритм — это итерационный процесс, который повторяет три основные операции 
(репродукция, кроссинговер, мутация) до достижения критерия остановки. В нашей реализации 
критерием остановки является достижение максимального числа поколений max_generations.

В этой работе реализовано два алгоритма 

1. Базовый простой написанный своими ручками 
2. Базовой на основе библиотеки deep 

## `class GeneticAlgorithmOptimizer`

`GeneticAlgorithmOptimizer` реализует стандартный генетический алгоритм с бинарным представлением хромосом. 
Оптимизация выполняется путем эволюционного поиска фазового распределения элементов метаповерхности с использованием 
операторов селекции, кроссинговера и мутации.

### Параметры конструктора

|----------|-----|--------------|----------|
| `config` | `MetasurfaceConfig` | - | Конфигурация метаповерхности (размеры M×N) |
| `population_size` | `int` | 200 | Размер популяции особей |
| `max_generations` | `int` | 300 | Максимальное количество поколений |
| `pc` | `float` | 0.85 | Вероятность кроссинговера (скрещивания) |

### Методы

#### `def optimize`

Метод запускает эволюционный процесс генетического алгоритма. На вход принимается целевая функция 

**Параметры:**
- `fitness_func` (`Callable`): Целевая функция, принимающая матрицу метаповерхности (M×N) и возвращающая скалярное значение, которое необходимо максимизировать.

**Возвращает:**
- `Tuple[np.ndarray, np.ndarray]`: Кортеж, содержащий:
  - `best_individual` - наилучший индивидуум (матрица фаз M×N)
  - `np.array(self.history)` - массив с историей изменения целевой функции на протяжении всей оптимизации

**Алгоритм работы:**

В цикле каждого поколения выполняется:

1. **Вычисление фитнес-функции** текущей популяции:
   ```python
   fitness_values = self.evaluate_population(population, fitness_func)
   ```

2. **Определение и обновление лучшего индивидуума**:
   ```python
   current_best_idx = np.argmax(fitness_values)
   current_best_fitness = fitness_values[current_best_idx]
   
   if current_best_fitness > best_fitness:
       best_fitness = current_best_fitness
       best_individual = population[current_best_idx].copy()
   ```

3. **Селекция, кроссинговер, мутация**:
   ```python
   parents = self.selection(population, fitness_values)
   children = self.crossover(parents)
   population = self.mutation(children)
   ```

#### `initialize_population() -> np.ndarray`

Инициализирует начальную популяцию случайными бинарными матрицами.

**Возвращает:**
- `np.ndarray`: Массив размером (population_size, M, N) со случайными значениями 0 и 1.

#### `evaluate_population(population: np.ndarray, fitness_func: Callable) -> np.ndarray`

Вычисляет значения приспособленности для всей популяции.

**Параметры:**
- `population` (`np.ndarray`): Текущая популяция особей
- `fitness_func` (`Callable`): Целевая функция для оценки

**Возвращает:**
- `np.ndarray`: Массив значений приспособленности для каждой особи

#### `selection(population: np.ndarray, fitness_values: np.ndarray) -> np.ndarray`

Реализует процесс селекции (репродукции) для выбора наиболее приспособленных особей.

**Описание:**  
Селекция — это процесс выбора наиболее приспособленных особей из текущей популяции для создания следующего поколения 
на основе значения их целевой функции. Особи с более высоким значением целевой функции имеют более высокую вероятность 
дать один или несколько потомков в следующем поколении

**Параметры:**
- `population` (`np.ndarray`): Текущая популяция
- `fitness_values` (`np.ndarray`): Значения приспособленности

**Возвращает:**
- `np.ndarray`: Выбранные родительские особи для создания следующего поколения

**Алгоритм:**
- Вычисляются вероятности выбора пропорционально значениям фитнес-функции
- Если суммарная приспособленность близка к нулю, вероятности распределяются равномерно
- Выполняется случайный выбор с возвращением на основе вычисленных вероятностей

#### `crossover(population: np.ndarray) -> np.ndarray`

Реализует оператор кроссинговера (скрещивания).

**Описание:**  
Простой одноточечный кроссинговер выполняется в два этапа:

1. **Этап 1**: Особи (родители) случайным образом образуют пары
2. **Этап 2**: Каждая пара строк подвергается кроссинговеру следующим образом:
   - Целая позиция `k` вдоль строки выбирается равномерно случайным образом между 1 и длиной строки
   - Две новые строки `A₁` и `A₂` создаются путем обмена всеми символами между позициями `k+1` и длиной строки

**Параметры:**
- `population` (`np.ndarray`): Текущая популяция

**Возвращает:**
- `np.ndarray`: Новая популяция после кроссинговера

**Особенности:**
- Кроссинговер применяется с вероятностью `pc`
- Точка разрыва выбирается случайным образом для каждой пары

#### `mutation(population: np.ndarray) -> np.ndarray`

Реализует оператор мутации.

**Описание:**  
В простом генетическом алгоритме мутация — это случайное (с небольшой вероятностью) изменение 
значения позиции строки. Для бинарного представления мутация заключается в инверсии бита (0→1 или 1→0).

**Параметры:**
- `population` (`np.ndarray`): Текущая популяция

**Возвращает:**
- `np.ndarray`: Новая популяция после мутации

**Особенности:**
- Каждый бит мутирует с вероятностью `pm`
- Мутация применяется независимо к каждому элементу каждой особи

## `class DeapMetasurfaceOptimizer`

Реализует эволюционный подход к оптимизации фазового распределения элементов метаповерхности. 
Алгоритм использует бинарное представление фазы (0 или 1) для каждого элемента и применяет 
стандартные генетические операторы: двухточечный кроссинговер и мутацию с инверсией бита.

### Параметры конструктора

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `config` | `MetasurfaceConfig` | - | Конфигурация метаповерхности (размеры M×N) |
| `population_size` | `int` | 200 | Размер популяции особей |
| `max_generations` | `int` | 300 | Максимальное количество поколений |
| `pc` | `float` | 0.85 | Вероятность кроссинговера (скрещивания) |
| `pm` | `float` | 0.05 | Вероятность мутации для каждого бита |

### Атрибуты

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `M` | `int` | Количество строк метаповерхности |
| `N` | `int` | Количество столбцов метаповерхности |
| `flat_size` | `int` | Общее количество элементов (M×N) |
| `toolbox` | `base.Toolbox` | Инструментарий DEAP для генетических операторов |

#### `optimize(fitness_func: Callable) -> Tuple[np.ndarray, np.ndarray]`

Запускает процесс оптимизации метаповерхности.

**Параметры:**
- `fitness_func` (`Callable`): Функция пригодности, принимающая матрицу метаповерхности размером (M×N) и возвращающая скалярное значение (максимизируемое).

**Возвращает:**
- `Tuple[np.ndarray, np.ndarray]`: Кортеж, содержащий:
  - `top` - оптимальная матрица фаз (M×N) с бинарными значениями (0/1)
  - Пустой список (зарезервировано для истории оптимизации)

### Алгоритм работы

1. **Инициализация**: Создается начальная популяция случайных бинарных матриц
2. **Эволюционный цикл** (на протяжении `max_generations` поколений):
   - **Скрещивание**: Двухточечный кроссинговер с вероятностью `pc`
   - **Мутация**: Инверсия бита с вероятностью `pm` для каждого элемента
   - **Оценка**: Вычисление фитнес-функции для каждого потомка
   - **Селекция**: Турнирный отбор (размер турнира = 3) для формирования следующего поколения
3. **Результат**: Возвращается лучшая особь из финальной популяции
